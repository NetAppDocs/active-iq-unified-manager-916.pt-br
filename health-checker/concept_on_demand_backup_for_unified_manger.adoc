= Backup sob demanda para o Unified Manager
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Você pode usar a interface de usuário do Active IQ Unified Manager para gerar backup sob demanda sempre que necessário.  O backup sob demanda permite que você crie instantaneamente um backup usando o método de backup existente.  O backup sob demanda não diferencia entre backup baseado em MySQL ou NetApp Snapshot.

Você pode executar backup sob demanda usando o botão *Fazer backup agora* na página Backup do banco de dados.  O backup sob demanda não depende dos agendamentos que você configurou para o Active IQ Unified Manager.
