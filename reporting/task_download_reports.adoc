= Baixar relatórios
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Você pode baixar relatórios e salvar os dados em uma unidade local ou de rede como um arquivo de valores separados por vírgula (CSV), um arquivo do Microsoft Excel (.XLSX) ou um arquivo PDF.  Você pode abrir arquivos CSV e XLSX com aplicativos de planilha, como o Microsoft Excel, e arquivos PDF com leitores como o Adobe Acrobat.

.Passos
. Clique no botão *Relatórios* para baixar o relatório como um dos seguintes:
+
[cols="2*"]
|===
| Escolher | Para... 


 a| 
Baixar CSV
 a| 
Salve o relatório como um arquivo de valores separados por vírgula (CSV).



 a| 
Baixar PDF
 a| 
Salve o relatório como um arquivo .pdf.



 a| 
Baixar Excel
 a| 
Salve o relatório como um arquivo do Microsoft Excel (XLSX).

|===
