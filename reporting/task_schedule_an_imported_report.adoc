= Agendar relatórios .rptdesign importados
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Você pode agendar relatórios existentes que foram criados e importados em uma versão anterior do Unified Manager.

O agendamento de relatórios importados requer o seguinte:

* relatórios de arquivo .rptdesign projetados pelo BIRT importados em uma versão anterior do Unified Manager
* aplicável ao atualizar para o Unified Manager 9.6 GA ou posterior


Após a atualização para o Unified Manager 9.6 GA ou posterior, a página Agendamentos de relatórios lista os relatórios importados.  Você pode editar o agendamento desses relatórios para especificar os endereços de e-mail dos destinatários, a frequência e o formato (PDF ou CSV).  Caso contrário, esses relatórios não poderão ser editados ou visualizados na interface do usuário do Unified Manager.

.Passos
. Abra a página Agendamentos de relatórios.  Se você tiver importado relatórios, uma mensagem será exibida.
+
image::../media/message_non_scehduled_reports.png[Uma captura de tela da interface do usuário que mostra a mensagem aparece se houver relatórios importados.]

. Clique no nome *Exibir* para exibir a consulta SQL que está sendo usada para gerar o relatório.
+
image::../media/importedreport1.png[Uma captura de tela da interface do usuário que mostra a consulta SQL usada para gerar o relatório.]

. Clique no ícone maisimage:../media/more_icon.gif[""] , clique em *Editar*, defina os detalhes da programação do relatório e salve o relatório.
+
[NOTE]
====
Você também pode excluir quaisquer relatórios indesejados no ícone maisimage:../media/more_icon.gif[""] .

====
