= Personalizar relatórios de VM de armazenamento
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Você pode criar relatórios de VM de armazenamento para analisar informações de volume e visualizar a integridade geral e a disponibilidade de armazenamento.  Por exemplo, você pode criar relatórios para exibir SVMs atingindo a contagem máxima de volume e analisar SVMs interrompidos.
