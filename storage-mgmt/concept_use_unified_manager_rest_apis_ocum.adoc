= Usar APIs REST do Unified Manager
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
O Active IQ Unified Manager fornece APIs REST para visualizar informações sobre monitoramento e gerenciamento do seu ambiente de armazenamento.  As APIs também permitem provisionar e gerenciar objetos de armazenamento com base em políticas.

Você também pode executar APIs ONTAP em todos os clusters gerenciados ONTAP usando o gateway de API suportado pelo Unified Manager.

Para obter informações sobre APIs REST do Unified Manager, consultelink:../api-automation/concept_get_started_with_um_apis.html["Introdução às APIs REST do Active IQ Unified Manager"] .
