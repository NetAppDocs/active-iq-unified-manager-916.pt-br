= Descrição das caixas de diálogo de cotas
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Você pode usar a opção apropriada na guia Cotas de usuário e grupo na exibição Integridade: todas as VMs de armazenamento para configurar o formato da notificação por e-mail que é enviada quando ocorre um problema relacionado à cota e para configurar regras para especificar endereços de e-mail com base na cota do usuário.
