= Como funciona o processo de descoberta
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Depois de adicionar o cluster ao Unified Manager, o servidor descobre os objetos do cluster e os adiciona ao seu banco de dados.  Entender como o processo de descoberta funciona ajuda você a gerenciar os clusters da sua organização e seus objetos.

O intervalo de monitoramento padrão é de 15 minutos: se você adicionou um cluster ao servidor do Unified Manager, levará 15 minutos para exibir os detalhes do cluster na interface do usuário do Unified Manager.
