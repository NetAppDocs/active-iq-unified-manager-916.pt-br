= Aplique filtragem para reduzir a lista de objetos correlacionados na grade
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
A filtragem permite que você exiba um subconjunto menor e mais bem definido de objetos na grade.  Por exemplo, se você tiver 25 volumes na grade, a filtragem permitirá que você visualize apenas os volumes com taxa de transferência menor que 90 MBps ou latência maior que 1 ms/op.
