= Descrição das janelas e caixas de diálogo de relacionamentos de proteção
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Você pode visualizar e gerenciar detalhes relacionados à proteção, como pools de recursos, associações de SVM e trabalhos de proteção.  Você pode usar a página Limites de integridade apropriada para configurar valores de limite de integridade globais para agregados, volumes e relacionamentos.
