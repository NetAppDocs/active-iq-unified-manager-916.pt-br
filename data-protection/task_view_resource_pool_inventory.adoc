= Ver inventário de pools de recursos
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Você pode usar a página Pools de recursos para visualizar o inventário do pool de recursos e monitorar a capacidade restante de cada pool de recursos.

.Antes de começar
Você deve ter a função de Administrador de Aplicativos ou Administrador de Armazenamento.

.Etapa
. No painel de navegação esquerdo, clique em *Proteção* > *Pools de recursos*.
+
O inventário do pool de recursos é exibido.
