= Remover a configuração do OnCommand Workflow Automation do Unified Manager
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Você pode remover a configuração do OnCommand Workflow Automation do Unified Manager quando não quiser mais usar o Workflow Automation.

.Antes de começar
Você deve ter a função de Administrador de Aplicativos ou Administrador de Armazenamento.

.Passos
. No painel de navegação esquerdo, clique em *Geral* > *Automação de fluxo de trabalho* no menu Configuração à esquerda.
. Na página *Automação de fluxo de trabalho*, clique em *Remover configuração*.
