= O que acontece quando o OnCommand Workflow Automation é reinstalado ou atualizado
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Antes de reinstalar ou atualizar o OnCommand Workflow Automation, você deve primeiro remover a conexão entre o OnCommand Workflow Automation e o Unified Manager e garantir que todos os trabalhos do OnCommand Workflow Automation em execução ou agendados sejam interrompidos.

Você também deve excluir manualmente o Unified Manager do OnCommand Workflow Automation.

Depois de reinstalar ou atualizar o OnCommand Workflow Automation, você deve configurar a conexão com o Unified Manager novamente.
