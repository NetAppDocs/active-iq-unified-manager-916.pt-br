= Navegadores suportados
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Para acessar a interface da Web do Unified Manager, use um navegador compatível.

A Matriz de Interoperabilidade tem a lista de versões de navegadores suportadas.

http://mysupport.netapp.com/matrix["mysupport.netapp.com/matrix"^]

Para todos os navegadores, desabilitar os bloqueadores de pop-up garante que os recursos do software sejam exibidos corretamente.

Se você planeja configurar o Unified Manager para autenticação SAML, para que um provedor de identidade (IdP) possa autenticar usuários, verifique também a lista de navegadores suportados pelo IdP.
