= Desinstalar produtos de terceiros
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Após atualizar o Unified Manager, você pode desinstalar produtos de terceiros, como Python ou Microsoft Visual C++ 2013. Isto é opcional.



[[uninstall-python]]
== Desinstalar Python

.Passos
. No *Painel de Controle* selecione *Programas* > *Programas e Recursos*.
. Selecione Python na lista de programas.
. Selecione *Desinstalar este programa*. Desinstale qualquer versão do Python, exceto a 3.12.  Certifique-se de manter a versão 3.12 do Python.




[[uninstall-microsoft-visual-c-2013]]
== Desinstalar o Microsoft Visual C++ 2013

.Passos
. No *Painel de Controle* selecione *Programas* > *Programas e Recursos*.
. Selecione *Microsoft Visual C++ 2013* na lista de programas.
. Selecione *Desinstalar este programa*.
